# QOMISO Game Tour

自転車ロードレースをモチーフにした育成・デッキ編成・レースバトル型ゲームアプリの初期プロトタイプです。

## コンセプト

- 個人育成モードで、主人公選手の瞬発力・登坂力・持久力・技術・連携を伸ばす
- サポートカードを組み合わせて、育成補正とレース中スキルを作る
- 5人チームを編成し、ステージ特性に合わせてレースバトルを解決する

## 現在の実装

- 静的HTML/CSS/JavaScriptのみで動作
- `index.html` をブラウザで開くとプレイ可能
- 育成12週、サポート4枚、チーム5人、4種類のステージを実装
