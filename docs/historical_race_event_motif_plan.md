# Historical Race Event Motif Plan

過去の有名レース、名勝負、伝説的な展開を研究用モチーフとして整理し、ゲーム内では架空イベント・架空コース・特殊ルールへ変換して使う。

## 基本方針

- 実在レース名、実在選手名、実在チーム名は開発用リファレンスに限定する。
- ゲーム本編では架空名へ変換し、出来事の構造だけを借りる。
- 歴史的事件、失格、ドーピング、事故、死亡事故などは直接再現せず、抽象化する。
- 名場面は「誰が勝ったか」よりも「どんな展開がゲームとして面白いか」で分類する。
- 1年シーズン、グランツール、ワンデークラシック、カード獲得イベント、スカウト演出の素材に分解する。

## イベント分類

| 分類 | ゲーム用途 | 例 |
|---|---|---|
| グランツール決戦 | 21ステージ内の山岳/TT/横風/最終日イベント | 最終TT逆転、山岳大逃げ、雨天山岳 |
| モニュメント | 最高峰ワンデー | 石畳、短坂、300km級、秋の山岳クラシック |
| クラシック名場面 | 1日レースの特殊条件 | 泥、横風、落車多発、少人数スプリント |
| 山岳伝説 | 山岳賞/総合逆転/ソロ逃げ | 超級山岳、悪天候、標高、補給失敗 |
| スプリント伝説 | 平坦ステージとポイント賞 | 発射台、混戦スプリント、位置取り戦 |
| TT伝説 | 個人TT/チームTT | 最終TT逆転、プロローグ、隊列崩壊 |
| 逃げ伝説 | ステージハンター用 | 長距離逃げ、総合勢の牽制、逃げ容認 |
| チーム戦術事件 | チームワーク/エゴ/エース変更 | 山岳列車、複数エース、アシストの犠牲 |

## ゲーム変換ルール

| 実史モチーフ | ゲーム変換 |
|---|---|
| 最終TTで総合逆転 | グランツール最終盤の高緊張TT。メンタルと技術が勝率に強く影響 |
| 石畳クラシック | 技術、負けん気、機材、位置取り、落車回避を重視 |
| 山岳大逃げ | スタミナ、登坂、エゴ、補給、チームの容認判断を重視 |
| 横風分断 | チームワーク、隊列、位置取り、ロードキャプテンの価値を上げる |
| 雨天/雪/寒さ | 負けん気、技術、機材選択、疲労回復に補正 |
| 少人数スプリント | 純粋な瞬発だけでなく、パンチャーやクラシック型にも勝機 |
| アシストの献身 | エース保護、ボトル運び、牽引、落車復帰のイベント化 |
| 複数エースの衝突 | エゴ、チームワーク、監督判断、レース中エース変更に接続 |

## 1年シーズンへの配置

- 2月: 砂漠/地中海の開幕ステージレース、初期調整イベント
- 3月: 長距離スプリントクラシック、石畳前哨戦
- 4月: 石畳モニュメント、アルデンヌ系短坂クラシック
- 5月: 第1グランツール、雨天山岳、逃げ屋イベント
- 6月: 総合調整ステージレース、各国選手権
- 7月: 最高峰グランツール、最終TT/山岳決戦/横風分断
- 8月: 第3グランツール、酷暑、山岳ステージハンター
- 9月: UCRチャンピオンシップ、TT特化イベント
- 10月: 秋の山岳クラシック、UCR年間ポイント最終評価

## 実装候補

- `historical_event_motifs.csv` を元に、架空イベント名を生成する。
- レースごとに `trigger_condition` を設定し、カードバトル中や年間スケジュールで発生させる。
- 名場面イベントはカード化できる。例: 「伝説の最終TT」「泥の石畳」「横風分断」「山岳大逃げ」。
- 選手500人データの脚質とイベント適性を接続し、特定モチーフ選手が得意な名場面を作る。

## 注意

- 実在の事故、違反、裁定は、ゲームでは「トラブル」「失速」「規定違反リスク」程度に抽象化する。
- 実名選手の名誉を損なう形でイベント化しない。
- 架空イベント名、架空地名、架空年代へ変換してから本編に出す。
