# チームカード設定

チームカードは個人カードとは別管理とし、全18種類から各ステージ用の6枚を編成する。ランダムドローは行わず、編成した6枚はレース中いつでも選択できる。

## 編成ルール

- 基本・得意・勝負の種類制限を設けず、全18種類から好きな6枚を選ぶ。同じカードは重複編成できない。
- 基本カードのコストは1、得意カードは2、勝負カードは3。
- 1ターンに使用できるチームカードは1枚まで。
- 勝負カードは1ステージにつき1回だけ使用できる。
- 個人カードとチームカードの合計作戦コストは5以内。
- ステージ間ではデッキを組み替えられる。

## 基本カード

| カード | 主効果 |
|---|---|
| エース保護 | エースの疲労と事故リスクを軽減 |
| 位置取り | 集団前方を確保し、分断と落車へ備える |
| 集団追走 | 逃げ集団とのタイム差を縮める |
| 集団補給 | 補給と健康度を回復する |
| ペース維持 | 疲労と乳酸の増加を抑える |
| 風よけ | 横風と向かい風の消耗を軽減 |
| 機材対応 | 機材トラブルの損失を軽減 |
| 下り統制 | 下りの落車リスクを抑えて集団を維持 |
| 逃げ容認 | 追走を控えてチームの体力を温存 |

## 得意カード

| カード | 主効果 |
|---|---|
| 山岳列車 | 山岳終盤までエースを保護 |
| スプリント隊列 | SPを温存し、フィニッシュを強化 |
| 横風分断 | 相手集団へエシュロンを仕掛ける |
| 逃げ形成 | 指定選手を有力な逃げへ送る |
| TTローテーション | TTの巡航速度を維持 |
| ダウンヒル攻勢 | 峠通過後の下りでタイム差を作る |

## 勝負カード

| カード | 主効果 |
|---|---|
| 総力牽引 | アシストの体力と引き換えにタイム差を一気に縮める |
| 二段攻撃 | 最初の攻撃を囮にして次の攻撃を強化 |
| エース交代 | サブエースへ護衛と勝負権を切り替える |

数値とブラウザ用カード定義は `data/team_card_catalog.js` を正本とする。
