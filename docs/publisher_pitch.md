# QOMISO Game Tour Publisher Pitch

## 1. Elevator Pitch

QOMISO Game Tour は、自転車ロードレースを題材にしたCredit年俸制デッキ編成・ターン制カードバトルゲームです。

プレイヤーは固定能力の選手カード8枚、サポートカード4枚、戦術カード最大5枚を編成し、当日エースと各役割を決め、平坦、丘陵、山岳、個人TT、チームTT、石畳、グラベル、横風などの局面をターン制で攻略します。

カードゲーム／デッキ構築の「編成読み合い」と「手札の切りどころ」を、ロードレース特有のチーム戦術へ落とし込むことが本作の核です。

## 2. Core Concept

- ジャンル: スポーツカードゲーム + デッキ編成 + ターン制レースバトル
- モチーフ: 自転車ロードレース、グランツール、クラシック、世界選手権、歴史的名場面
- プレイ体験: カード獲得、年俸制デッキ編成、レースごとの役割設定、戦術カード選択、ターン制バトル
- 想定プラットフォーム: Mobile first / PC展開可
- 想定セッション: 1レース 2-3分、1ステージ重要局面制、グランツールは長期目標

## 3. Unique Selling Points

### ロードレースのチーム戦術をゲーム化

一般的なスポーツゲームはスター選手を並べるほど強くなりがちですが、本作ではエースだけを集めても勝てません。

- アシストがいないと補給、位置取り、横風、石畳、山岳列車、チームTTで崩れる
- ロードキャプテンがいると事故、分断、エース変更に強くなる
- リードアウトがいるとスプリント勝率が上がる
- Credit年俸制により、高額エース偏重を自然に制限する

### Credit年俸制

選手価値はランク文字ではなく Credit年俸 で表します。

- 世代最強級の例外枠: 15000 Credit
- 歴代級・最上位エース: 11000-12000 Credit
- トップエース: 9000-10999 Credit
- 主力: 6000-8999 Credit
- 中核／専門アシスト: 3000-5999 Credit
- 低コスト／専門役割カード: 800-2999 Credit

8人チームの総年俸上限は、通常レース42,000Cr、上位レース45,000Cr、最高峰レース48,000Cr。プレイヤーはレース格ごとに異なる予算で「勝てる構造」を作る必要があります。

### 重要地点ターン制レース

リアルタイム操作ではなく、レースの勝負所だけをターン化します。

- 山岳入口
- 横風区間前
- 石畳/グラベル入口
- 中間ポイント前
- 補給後
- 残り局面
- フィニッシュ

各ターンでカード/アビリティを選び、シミュレーション結果と次の展開が提示されます。

### 実在モチーフを架空化した厚い世界観

実在選手、チーム、レース、機材、歴史的名場面を研究用モチーフとして分析し、ゲーム内では架空名・架空設定へ変換します。

- 研究用500人候補CSV
- 名アシスト/ロードキャプテン/リードアウト候補CSV
- 歴史的レースイベント40件の初期モチーフ
- 架空22チーム
- 架空グランツール/クラシック/TTT

- トレーディングカード、デッキ構築、役割シナジーが好きな層

- スポーツマネジメントゲームのファン

## 5. Game Loop

1. 年初にチーム方針を決める
2. Credit年俸上限内で8人チームを編成する
3. 機材とウェアを選ぶ
4. 選手・サポート・戦術カードでレースデッキを完成させる
5. 年間カレンダーからレースへ出場する
6. 当日のエースとアシスト役を決める
7. 重要地点ターン制バトルを行う
8. 勝利、順位、疲労、イベント、契約価値が変動する
9. シーズンを進め、グランツール/クラシック制覇を目指す
10. UCR年間ポイントを獲得し、ライダー／チーム／マネージャー年間王者を目指す

## 6. Race Battle

1回のレースバトルは2-3分を目標にします。

200km級のワンデーレースでも、全距離をリアルタイムで操作するのではなく、重要地点のみを選択ターンにします。

### バトル要素

- 選択カード: 位置取り、牽引、アタック、エース保護、スプリント準備
- 地形: 平坦、丘陵、山岳、個人TT、チームTT
- 条件: 風、グラベル、石畳、雨、寒さ、補給、落車リスク
- メンタル: チームワーク、エゴ、負けん気
- 編成: エース、アシスト、ロードキャプテン、リードアウト

## 7. Season Structure

年間カレンダーは国際ロードサイクル連合（UCR）が公認する「UCR World Circuit」とし、全公認レースからUCRポイントを獲得する。

### ワンデー

- 石畳クラシック
- グラベルクラシック
- 丘陵クラシック
- スプリントクラシック
- 秋の山岳クラシック
- UCRチャンピオンシップ

### ステージレース

- 7日級ステージレース
- 10日級ステージレース
- 21ステージの最高峰グランツール

グランツールは省略しません。ただし、プレイヤーが重要でないと判断したステージは自動シミュレートで進行でき、疲労、事故、順位、ポイント変動は処理されます。

## 8. Monetization

本作は「勝つために課金する」よりも、「チーム作りの幅を増やす」課金を目指します。

### 想定収益構成

| 収益軸 | 役割 |
|---|---|
| シーズンパス | 年間シーズン進行、カードスリーブ、称号、ウェア、演出 |
| 緩いスカウト | 選手候補、スタッフ、若手、サポート。Credit年俸制で価値管理 |
| 見た目課金 | ウェア、バイクカラー、チームバス、UIテーマ |
| 拡張パック | 架空グランツール、クラシック、歴史的イベントパック |

### Avoid

- ランク文字による強さ煽り
- レベル上げ、限界突破、重複カードによる選手強化
- 期間限定人権
- 機材性能の有料抽選
- PvP勝利に直結する課金限定能力
