# QOMISO Game Tour One Page Pitch

## 一言

自転車ロードレースを題材にした、Credit年俸制デッキ編成・ターン制カードバトルゲーム。

## 何が新しいか

- エースだけを集めても勝てないスポーツゲーム
- アシスト、ロードキャプテン、リードアウトが勝敗を左右する
- 選手価値はランク文字ではなく Credit年俸
- 8人チームをレース格別の総年俸上限内で編成する（通常42,000Cr／上位45,000Cr／最高峰48,000Cr）
- 200kmのレースを2-3分の重要地点ターン制バトルに圧縮
- 実在ロードレースの歴史、選手、チーム、機材、名場面を架空化して厚い世界観を作る

## ゲームループ

カード獲得 -> 選手8枚＋サポート4枚＋戦術5枚を編成 -> 当日役割設定 -> 重要地点カードバトル -> UCR年間ポイント -> グランツール/クラシック制覇

国際ロードサイクル連合（UCR）の年間ポイントを競い、非同期対人リーグでは年間実績ポイントと実力レートを分けて運用する。

## 主要システム

- 固定能力の選手カードと主副脚質／役割適性
- サポートカード＋戦術カードデッキ
- Credit年俸上限
- アシスト不足/エース過多ペナルティ
- 平坦、丘陵、山岳、個人TT、チームTT
- 風、石畳、グラベル
- 21ステージを省略しないグランツール
- 自動シミュレートでも疲労/事故/順位/ポイントを処理

## 収益化

- シーズンパス
- 緩いスカウト
- ウェア、バイクカラー、UIテーマなどの見た目課金
- 架空グランツール/クラシック/歴史的イベント拡張パック

避けるもの: レベル上げ、限界突破、重複強化、期間限定人権、機材性能の有料抽選、PvP勝利に直結する課金限定能力。
